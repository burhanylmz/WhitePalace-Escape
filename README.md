🏰 Whitepalace Escape
Whitepalace Escape, Unity 6 motoru ile geliştirilmiş, 16-bit estetiğine sahip bir 2D pixel-art platform oyunudur. Oyuncu, gizemlerle dolu beyaz saraydan ve onun farklı diyarlarından kaçmaya çalışan bir şövalyeyi kontrol eder.

🚀 Temel Özellikler
Farklı Biyomlar: Yeşil Orman, Kavurucu Çöl, Buzul Dağlar ve Gizemli Balkabağı temalı 4 ana bölüme sahip harita yapısı.

Gelişmiş Mekanikler: Hassas kontrol sağlayan "Uzun Zıplama" sistemi ve düşmanlardan kaçış odaklı oynanış.

Bölüm Seçme Sistemi: Oyuncuların açtıkları haritalar arasında geçiş yapabildiği interaktif ana menü.

Dinamik Ses Sistemi: Singleton tasarımı ile sahneler arası kesilmeyen arka plan müziği ve aksiyon odaklı ses efektleri.

🛠 Kullanılan Teknolojiler
Oyun Motoru: Unity 6.

Programlama Dili: C#.

Grafik: Pixel Art Estetiği (Fantasy Theme).

Versiyon Kontrol: Git / GitHub.
